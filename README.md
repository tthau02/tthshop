E-Commerce Website

Dự án gồm 2 phần: **Frontend (fe)** và **Backend (be)**. Hệ thống hỗ trợ chức năng quản trị tại `/admin`.

## 🏁 Bắt đầu nhanh

1. Cài đặt Backend

cd be
npm install
npm run dev

2. Cài đặt Frontend

cd fe
npm install
npm run dev
